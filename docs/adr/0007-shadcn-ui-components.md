# shadcn/ui as the component library, default zinc theme

UI components are built with shadcn/ui using its default zinc/neutral theme. shadcn copies component source directly into the repo (no runtime dependency), which means components are fully customisable without fighting a library's abstractions. Built on Radix UI primitives for accessibility.

Chosen over Mantine and MUI because it ships no runtime bundle, integrates with Tailwind v4, and its default aesthetic matches the existing zinc palette already in use.
